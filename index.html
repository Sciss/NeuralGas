DemoGNG 2.1
This java applet implements several methods related to competitive learning and self-organizing "neural" networks. Try the "Random" button initially. The applet can also be started outside the browser via webstart (see Launch button below) to avoid current security considerations with the java plugin for Java 7 Update 11.
